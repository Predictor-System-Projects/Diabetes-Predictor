# Diabetes-Predictor
Made using Support Vector Machine Algorithm (SVM) in Python.  
This Model predicts if a person is Diabetes Positive or Negative based on the following Data entered:-   
1) Glucose.  
2) BloodPressure.  
3) SkinThickness. 
4) Insulin.  
5) BMI.  
6) DiabetesPedigreeFunction.  
7) Age.  
8) Outcome.
